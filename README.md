# Climb Insight Hub

Aplicação React + Vite para a interface interna da Climb.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`

## Estrutura

- `src/pages`: páginas de rota
- `src/components/login`: elementos reutilizáveis das telas de autenticação
- `src/components/ui`: componentes de interface compartilhados
- `src/hooks`: hooks da aplicação
- `src/lib`: utilitários
