# Climb App

Frontend do fluxo de acesso da Climb, construído com React, TypeScript, Vite e Tailwind CSS.

## Requisitos

- Node.js 22+
- npm 10+

## Como executar

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Estrutura

```text
src/
  components/login/   componentes visuais da marca e ilustração lateral
  hooks/              hooks de interface, como alternância de tema
  pages/              telas de acesso, recuperação de senha e solicitação
  App.tsx             roteamento principal
```
